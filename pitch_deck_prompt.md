### 第 6 页 — 核心技术③：意图预检测

- **标题**：核心技术③ — 意图预检测（3-5x 响应加速）
- **内容要点**：
  - 行业痛点：传统 Agent 依赖 LLM 判断是否调用工具，存在 2-5 秒额外推理延迟、判断不稳定
  - 解决方案：在 Agent 层之前加入中英文关键词意图预检测层
  - 工作流程：用户输入 → 关键词匹配（<5ms）→ 命中则直接执行工具（跳过 LLM 推理）→ 未命中走常规 Agent 流程
  - 覆盖操作：出题 / 闪卡 / 诊断 / 速查 / 播客 — 5 类高频操作
  - 效果：命中率 100%（关键词精确匹配），响应速度提升 3-5 倍
