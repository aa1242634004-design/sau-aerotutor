### 第 4 镜 | 1:05 — 1:35 | 出题测验（意图预检测核心展示）🔥

| 项目 | 内容 |
|------|------|
| **画面** | 点击快捷按钮「📝 出题测验」→ 立刻弹出测验卡片（基于论文内容的 3 道题：选择 + 简答）→ 用户勾选一道选择题 → 即时判对错 → 输入一道简答 → AI 打分 |
| **鼠标** | 1) 点击快捷按钮「出题测验」<br>2) ~1s 后测验卡片弹出<br>3) 选择题点击某个选项 → 即时变绿/红判定，显示解析<br>4) 简答题输入一句话答案 → 提交 → 显示 AI 逐条评分 |
| **关键帧** | 测验卡片弹出瞬间，右侧叠加文字：「⚡ 意图预检测：直接执行，跳过 LLM 推理」 |
| **旁白** | "点击出题测验——系统通过意图预检测，直接调用出题引擎，不需要 LLM 判断"是否该出题"。响应速度提升 3 到 5 倍。选择题即时判对错，简答题 AI 按要点打分。得分低于 60% 自动收录错题本。" |

> 🎬 **全片核心亮点**：测验卡片出现时加 0.5s 轻微缩放动画 + 清脆"叮"音效。建议叠加半透明对比标注：「传统 Agent：等 LLM 决定要不要出题（3-5s）→ AeroTutor：关键词直接命中（<5ms）」

## 额外建议

1. **论文页数多不是问题** — 25 页 PDF 解析需要 5-10 秒，提前上传好，录制时只展示结果
2. **论文标题在视频中展示** — 第 2 镜中 PDF 文件名给 1-2 秒特写，让评委看清是真实 arXiv 论文
3. **中英混合是亮点** — 英文论文 + 中文问答，恰恰展示了系统对中英混合场景的适配能力
4. **第 4 镜的对比标注** — "传统 Agent vs AeroTutor"的对比是说服评委的关键，建议用半透明深色底 + 白色文字，居中偏下放置，持续 5 秒
5. **准备备用回答** — 如果 DeepSeek API 响应质量不稳定，提前录几组回答选最好的那条用
